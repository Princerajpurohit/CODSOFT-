# CodeSoft Internship – Python Task Portfolio

This repository contains the projects I developed during my **CodeSoft Internship**.
The purpose of these tasks was to practice problem-solving, strengthen programming logic, and build small real-world applications using Python.

Each project focuses on a specific concept such as user interaction, conditional logic, randomness, and GUI development.

---

## 🛠️ Tech Stack

**Language:** Python 3.x
**Libraries Used:** Tkinter, Random, OS, JSON
**Tools:** VS Code, Git, GitHub

---

## 🚀 Projects Overview

### 1️⃣ To-Do List Application 📝

A **GUI-based task management application** designed to help users organize their daily activities.

**Features**

* Add new tasks
* Update existing tasks
* Delete tasks
* Mark tasks as completed
* Simple graphical interface using Tkinter

**Key Learning**

* GUI development using Tkinter
* Handling lists and task data
* Managing user input through graphical elements

---

### 2️⃣ Calculator 🔢

A **CLI (Command Line Interface) based calculator** that performs basic arithmetic operations directly in the terminal.

**Features**

* Addition
* Subtraction
* Multiplication
* Division
* Error handling for invalid inputs

**Key Learning**

* Conditional logic and control flow
* Working with terminal input/output
* Implementing basic arithmetic operations

---

### 3️⃣ Password Generator 🔑

A **GUI-based security utility** that generates strong and random passwords.

**Features**

* Custom password length
* Option to include numbers and special characters
* Random password generation
* User-friendly graphical interface

**Key Learning**

* Using the `random` and `string` libraries
* Implementing GUI components with Tkinter
* Building small security-focused tools

---

### 4️⃣ Rock-Paper-Scissors Game 🎮

A **GUI-based interactive game** where the user plays against the computer.

**Features**

* Graphical buttons for user choices
* Random computer decision-making
* Score tracking
* Interactive interface

**Key Learning**

* Game logic implementation
* Randomized computer moves
* Designing interactive GUI applications

---

## 📌 Project Structure

* **Calculator:** CLI-based application
* **To-Do List, Password Generator, Rock-Paper-Scissors:** GUI-based applications built using **Tkinter**

---

## 📌 Conclusion

This internship helped improve my **Python programming skills, logical thinking, and practical problem-solving abilities**.
Through these projects, I gained experience in both **Command Line (CLI) applications and Graphical User Interface (GUI) development**, building functional tools and interactive programs.
